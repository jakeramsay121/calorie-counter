# calorie-counter
 Final Project

Overview:

The Calorie Couner is an application designed to help users track there daily food intake by manually entering food items and there corresponding calorie counts. Once data entry is complete, the application wil display a summary of total calories, average calories per item, and the food items with the highest and lowest calorie counts.

Files Include: 
1. main.py - The main driver of the program, handling user interaction.
2. calorie_counter.py - A module for managing calorie data and computing statistics.

To Run Calorie Counter:
1. Download the project files to a directory on computer.
2. Open a terminal.
3. Navigate to the directory containing project files.
4. Run the program as directed.
